Welcome to Gyrus Framework

What is Gyrus Framework ?

Gyrus is a RAD Framework to develop HTML5 RDBMS-backed Business Applications

Background

We're developing large business application since 1999. This is an implementation of our test & tried methodologies with an HTML5 Stack. Previous implementations were COM & NET.

Opinionated

In the realm of large business applications. We know what works, what does not, and why. So, because we have a specific goal, this is a very opinionated framework.

Stack

DB Server (3rd Tier): Almost ANY RDBMS (Objective) - Postgre - MariaDB - ORACLE

WEB (2nd Tier, Application) Server: - NGINX - PHP

Front-End (1st Tier) Browser: - HTML5 - Javascript - Gr-Client (js) - AJAX

Developer Stack: - WPN-XM (Windows PHP NginX, XDebug,XPhrof, MariaDB - NetBeans (IDE, debugger for PHP 2nd tier code) - Chrome Develeper Tools (IDE, debug & continue for javascript 1st tier code, CSS designer)

Future (Alternative) Stack

WEB (2nd Tier, Application) Server: - NGINX - node.js
