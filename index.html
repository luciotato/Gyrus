Welcome to Gyrus

What is Gyrus Framework ?

Gyrus is a RAD Framework to develop State-of-the-Art HTML5 RDBMS-backed Business Applications

Background

We're developing Large Organization Applications since 1999. This is an implementation of our test & tried methodologies with a HTML5 Stack. Previous implementations were COM & NET.

Opinionated

In the realm of large business applications. We know what works, what does not, and why. Because we have specific goals, this is a very opinionated framework.

Stack

Front-End (1st Tier) Browser:

  • HTML5
  • Javascript
  • Gr-client
  • JQuery
  • AJAX

WEB Server (2nd Tier, Application Server):

  • NGINX
  • Gr-dataServer
  • PHP

DB Server (3rd Tier): RDBMS

  • Postgres
  • MariaDB
  • ORACLE
  • MS-SQL
  • MySQL

Developer Stack:

  • Nginx - PHP - XDebug+XPhrof - MariaDB/Postgres/ORACLE/MSSQL/MySQL
  • NetBeans (IDE, debugger for PHP 2nd tier code)
  • Chrome Develeper Tools (IDE, debugger, Edit&Continue for javascript 1st tier code, CSS designer)

Future (Alternative) 2nd Tier Stack

WEB Server (2nd Tier, Application Server):

  • NGINX
  • Gr-dataServer
  • node.js
